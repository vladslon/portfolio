## Overview

In this module, students will be adding to the skills they have already learned, and be introduced to more in-depth CSS.

  * Git/GitHub

  * HTML
  * CSS Flexbox
  * CSS Grid
  * Media Queries
  * Wireframing
  * CSS selectors
  * CSS variables
